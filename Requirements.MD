Task: Employee Management System
Build a simple Employee Management System using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

Features Required
Employee Management
Add Employee

View Employee List

Edit Employee Details

Delete Employee

Employee Fields
Profile Image Upload

Name

Email

Mobile Number

Department

Technical Requirements
Create REST APIs using Node.js and Express.js.

Store employee data in MongoDB.

Implement profile image upload functionality.

Connect the React frontend with backend APIs.

Implement proper form validation.

Use clean and organized code structure.

Deployment Requirement
Deploy the frontend and backend application on Render, Vercel, or any similar cloud platform.

Ensure the application is publicly accessible and fully functional.
